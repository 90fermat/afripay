# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AfriDevPay, please **do not** open a public GitHub issue.

Instead, report it privately via:
- **Email**: security@afripay.dev
- **GitHub**: Use the [private vulnerability reporting](https://github.com/90fermat/afripay/security/advisories/new) feature

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

In scope:
- API authentication bypass
- SQL injection / data exposure
- Payment data leakage
- API key exposure

Out of scope:
- Issues in third-party provider SDKs
- Rate limiting bypass in sandbox/test environment
- Social engineering

Thank you for helping keep AfriDevPay secure.
